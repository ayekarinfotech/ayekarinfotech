## 🏢 Who We Are

Ayekar Infotech is a software company based in Maharashtra, India, focused on building reliable, production-grade technology. We specialize in payment systems where correctness is non-negotiable, and we bring the same engineering discipline to everything else we build — from AI solutions to web and mobile applications.
